## CHEF APP
## Description
MyMobileApp is a mobile application made to help users manage their daily tasks easily. It has a clean design, quick navigation, and smart reminders to make everyday organization simple and efficient.
## Features
Easy navigation and user-friendly interface.
Notifications and reminders for important tasks.
Cloud sync to keep data safe and accessible.
Dark mode for comfortable night use.
## Technology Used
React Native for app development.
Firebase for backend and database.
Figma used for design and layout.
